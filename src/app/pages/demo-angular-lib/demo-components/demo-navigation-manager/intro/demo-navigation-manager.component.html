Medes Navigation Manager Components

Use page navigation manager components for paging items & set how many items will shown on the page. There are 2 categoires:

  • Filter Page Navigation Component <mds-page-navigation ... ></mds-page-navigation>
  • Filter Page Counter Component <mds-page-counter ... ></mds-page-counter>

Put sample data & configuration to component. Put configuration range start-end to splice array.
