Medes Filter Components

Multiple filter checkbox components is automatically generate HTML Element filter per group base on sampledata and props/keys & support url query param. There are 3 categoires:

  • Filter Checkbox Component <mds-filter-checkbox ... ></mds-filter-checkbox>
  • Filter Swatch Component <mds-filter-swatch ... ></mds-filter-swatch>
  • Filter Range Slider Component <mds-filter-range-slider ... ></mds-filter-range-slider>
  • Reset Filter Component <mds-filter-reset ... ></mds-filter-reset>

Put sample data, key to filter and selected filter object. Put selected filter object to *ngFor loop to filter pipe from medes-ui.
