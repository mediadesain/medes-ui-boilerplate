Grid classes list

Class nameDescription
.mds-grid Container of grid
.g-l-* Grid item size when screen is large like on laptop or monitor display (more than 768px)
.g-m-* Grid item size when screen is medium like tablet (more than 480px & less than 768px)
.g-s-* Grid item size when screen is small like on mobile phone (less than 480px)

Note: For * sizinng starting from 20 until 100, stepping 5 per grid item.
