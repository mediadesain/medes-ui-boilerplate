How to Install

Instalation step.

Follow these step to install MedesUI Library on your project.

  1. Make sure node.js installed on your computer.
  2. Select or cd project want to instal the library. Run on terminal/cmd npm install @mediadesain/core.
  3. Import all of medes-ui stylesheets by following these steps:
    • Import medes-ui.css or medes-ui.scss and also ui-colorscheme.css or ui-colorscheme.scss stylesheet on main global styling.
      CSS SCSS
    Documentation for creating your own color scheme click here.
  4. Medes Styling Element & Utils Helper script are ready to use.

Install MedesUI Libraries to your Project

  1. If will using Angular library like component, service, pipe. Install with running on terminal/cmd npm install @mediadesain/angular.
  2. If will use Ract libraries like component & others. Install with running on terminal/cmd npm install @mediadesain/react.
