Follow these step to install MedesUI Library on your project.

  1. Make sure node.js installed on your computer.
  2. Select or cd project want to instal the library. Run on terminal/cmd npm install medes-ui.
  3. Import all of medes-ui stylesheets by following these steps:
    • Import medes-ui.scss & ui-colorscheme.scss stylesheet on main styles.scss.
    • Or medes-ui.scss & ui-colorscheme.scss to angular.json.
    Documentation For creating your own ui-colorscheme.scss click here.
  4. Import modules, components or utils to use.
  5. Run with ng serve or npm start
