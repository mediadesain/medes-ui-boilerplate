Add Custom Color Scheme

For adding custom color scheme make sure already import medes-ui.scssa & ui-colorscheme.scss to main style.scss as per installation step. Open the style.scss itself. Add new styling file example custom-color.scss / custom-color.css on ./assets/style/custom-scheme.scss. Then Customize your own color scheme and we provide color generator below. And put these color swatch variable code on light & dark mode.

style.scss custom-color.scss
